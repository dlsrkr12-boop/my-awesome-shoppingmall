# Instagram Clone - React App

리액트로 만든 인스타그램 스타일 앱입니다.

## 📋 사전 준비

이 프로젝트를 실행하기 전에 다음 파일들이 필요합니다:

### 이미지 파일 구조
```
instagram-app/
  └── public/
      └── img/
          ├── img01.jpg
          ├── img02.jpg
          ├── img03.jpg
          ├── img04.jpg
          ├── img05.jpg
          ├── img06.jpg
          ├── img07.jpg
          ├── img08.jpg
          ├── img09.jpg
          ├── img10.jpg
          ├── man.png
          └── woman.png
```

**중요**: `public/img/` 폴더를 만들고 위의 이미지 파일들을 복사해주세요!

## 🚀 설치 및 실행

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm start
```

3. 브라우저에서 `http://localhost:3000` 자동 실행

## 📦 빌드

프로덕션 빌드:
```bash
npm run build
```

## ✨ 주요 기능
